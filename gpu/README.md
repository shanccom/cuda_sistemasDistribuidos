# Implementación GPU CUDA

## Descripción

Contiene la implementación acelerada mediante GPU utilizando CUDA.

## Objetivo

Reducir los tiempos de procesamiento aprovechando el paralelismo masivo de la GPU.

## Tecnologías

- CUDA
- CuPy
- OpenCV

## Funcionalidades

- Transferencia de datos CPU-GPU
- Procesamiento paralelo
- Medición de rendimiento
- Comparación con CPU

## Archivo principal

process_gpu.py

## Ejecución

```bash
python process_gpu.py
```
