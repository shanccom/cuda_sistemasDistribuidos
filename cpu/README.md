# Implementación CPU

## Descripción

Contiene la versión secuencial del procesamiento de imágenes médicas ejecutada en CPU.

## Objetivo

Servir como referencia para comparar el rendimiento frente a la implementación CUDA.

## Funcionalidades

- Lectura de imágenes
- Conversión a escala de grises
- Aplicación de filtros
- Medición de tiempo de ejecución

## Archivo principal

process_cpu.py

## Ejecución

```bash
python process_cpu.py
```
